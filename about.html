About Project

This project is a user management system built using HTML, CSS, JS, Bootstrap, and PHP. It allows the admin to create, update, and view users with their images.

The project uses PHP APIs to handle user creation, updating, and retrieval. It also incorporates session-based authentication to ensure secure access to the admin features.

Key features of the project include:

  • Admin login using session authentication
  • Ability to create new users with their images
  • Update user details such as name, email, and avatar
  • View a list of all users with their details

The project leverages Bootstrap for responsive design and a clean user interface. JavaScript is used for client-side interactions and form validations.

The combination of these technologies allows for a robust user management system that can be easily extended and customized as per specific requirements.
